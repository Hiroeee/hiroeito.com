Daily UI challenge

A series of daily design challenges - UI design

User Interface plays a crucial role in creating digital products.
After completing my Google UX design professional certificate course, I decided to commit to the Daily UI challenge to improve my interface skills. It aims to practice, consume inspiration, and master using the design tool daily.
Daily UI is a platform that sends a series of tasks every day regarding the user interface.

Disclaimer:

I am aware of the importance of the UX design process which must be constantly tested and backed by rigorous research for creating a user-centric design.
My UI challenge journey came to more than half. So far, it's been helpful in improving my overall design skills. Not only the visual techniques, but I also learned better time management and flexibility through the series of tasks. Those boost my confidence as a designer.
